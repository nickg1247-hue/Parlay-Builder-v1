Install NTG Sports

Add the site to your home screen — no app store required. You get a full-screen icon and faster loads.

iPhone / iPad (Safari)

  1. Open ntgsports.com in Safari (not Chrome).
  2. Tap the Share button (square with arrow).
  3. Scroll and tap Add to Home Screen.
  4. Tap Add — the NTG icon appears on your home screen.

Android (Chrome)

  1. Open ntgsports.com in Chrome.
  2. Tap the menu (⋮) or use the Install app banner when it appears.
  3. Choose Install app or Add to Home screen.
  4. Confirm — NTG Sports opens like a native app.

Live picks and odds always need an internet connection. Installing does not create a separate account — use the same sign-in as the website.
